🐍 Roadmap Belajar Python SMA

Dari Nol Sampai Bikin Project Keren! 🚀

⏱️ Estimasi waktu: 3-6 bulan • 5-7 jam/minggu
1

🐣 Level 1: Python Pemula

⏰ 3-4 minggu • 15-20 jam
  • Apa itu Python? Install & setup
  • Print & Comments (cetak teks)
  • Variabel & Tipe Data (string, integer, float)
  • Input dari user
  • Operator matematika dasar
  • String manipulation (menggabung, memotong)
  • List & Tuple (kumpulan data)
  • Dictionary (data berpasangan)
#print() #variabel #tipe_data #list

✅ Bisa bikin: Program salam, kalkulator sederhana

2

🧠 Level 2: Logic & Control

⏰ 4-5 minggu • 20-25 jam
  • If, Elif, Else (percabangan)
  • Perulangan For & While
  • Break & Continue
  • Fungsi (def) - bikin kode sendiri
  • Parameter & Return value
  • List comprehension (cara cepat bikin list)
  • Error handling (try-except)
#if_else #looping #fungsi #try_except

✅ Bisa bikin: Tebak angka, game sederhana, kalkulator canggih

3

⚡ Level 3: Intermediate

⏰ 5-6 minggu • 25-30 jam
  • Modul & Package (import)
  • Baca & tulis file (txt, csv)
  • OOP sederhana (class & object)
  • Lambda & Map/Filter
  • Working with dates
  • Regular Expression (regex) dasar
  • Virtual environment (venv)
#import #file_io #class #regex

✅ Bisa bikin: To-do list app, data organizer, password generator

4

🚀 Level 4: Project Based

⏰ 6-8 minggu • 30-40 jam
  • Web scraping dengan BeautifulSoup
  • Buat API sederhana (Flask)
  • Database SQLite
  • Buat bot Telegram/WhatsApp
  • Data visualization (matplotlib)
  • Game sederhana (pygame)
  • Deploy ke internet (pythonanywhere)
#flask #scraping #database #deploy

✅ Bisa bikin: Website sendiri, bot, aplikasi database
